displays a video or audio player that fits at most in its parent block

{up media-plyr=yt | code=bTqVqk7FSmY }
{up media-plyr=vimeo | code=bTqVqk7FSmY }
media-plyr : specify yt or youtube or vimeo
code : this is the code in the URL of the video.

{up media-plyr=video | poster=url image | mp4 | webm | vtt | download }
media-plyr : video to indicate the URL to the video.
poster : name of the still image
mp4, webm, vtt, download : if URL not specified, it will be deducted from the one to'post'.

{up media-plyr=audio | mp3=url mp3 file | ogg}
media_plyr : audio
mp3 : url to MP3 file (required)
ogg : if URL not specified, it will be deducted from the one for'MP3'.
