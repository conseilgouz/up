displays a random image

The Flickr API is closed as well as the alternatives.
To preserve existing shortcodes, the action generates an image with a colored background and the specified dimensions.

Syntax : {up lorempixel=type | width=xx | height=xx }
--> type = abstract, animals, business, cats, city, food, nightlife, fashion, people nature, sports, technics, transport
Note : width & height are the dimensions of the image returned by lorempixel. To display it in filling in the parent block, you must add style=width:100%
