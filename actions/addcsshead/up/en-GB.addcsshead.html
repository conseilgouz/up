adds CSS code or files to the head.

without risk of cleaning by a WYSIWYG editor
syntax 1 : {up addCssHead=.foo[color:red]} Be careful: put [ ] instead of {}
syntax 2 : {up addCssHead} .foo{color:red} {/up addCssHead}
syntax 3 : {up addCssHead=path}
Use : upload a page specific CSS file
