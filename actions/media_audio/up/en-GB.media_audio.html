displays a player (HTML5) for one or more audio files hosted on the server

just indicate the path and name of the file (with or without extension) to use all audio files of that name in the folder.
If the file name is not specified or contains wildcards, all matching audio files will be returned.
Support audio formats: mp3,ogg
If an image (jpg,png,webp,gif) exists, it will be used as a preview (poster)

syntax {up media-audio=path_to_audios}
Keywords for the template:
##player##, ##title##, ##image##, ##info##, ##download##
