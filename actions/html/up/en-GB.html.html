allows you to create an HTML entity (tag) with class(es), style and attribute without switching to mode code

Syntax 1 : {up html=div | class=foo | id=x123}CONTENT{/up html} --> < div id=x123 class=foo>CONTENT< /div> Syntax 2 : {up html=img | class=foo;border:1px red solid | src=images/img.jpg} --> < img class=foo style=1px red solid src=images/img.jpg > note: all options are considered attributes of the tag Syntax 3 : {up html=h1.foo.xx} equals {up html=h1 | class=foo xx}
